🗺️ Google Maps API Key ✓ Saved

Unlocks the draggable 🧍 Pegman and full Street View on the Street View Explorer page.

1. Go to console.cloud.google.com → your project
2. Enable Maps Embed API
3. Credentials → Create API Key
4. Restrict to: cashfrontandback.vercel.app

📧 Gmail OAuth Client ID ✓ Saved

2. Enable Gmail API
3. Credentials → Create OAuth Client ID → Web Application
4. Add to Authorized JavaScript origins
5. Paste your Client ID below

📥 Inbound Webhook — Receive Leads

POST lead data to this endpoint from your website, Zillow, Facebook Ads, or any external tool. Leads appear instantly in the CRM.

Endpoint URL
